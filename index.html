Comparisons

Less than or equal to is now represented as <= instead of as <.

< is now the less than operator which was missing on Roll20.

Greater than or equal is now represented as >= instead of as >.

> is now the greater than operator which was missing on Roll20.

Power

Rasing to a power can now be done using ^ as well as **
